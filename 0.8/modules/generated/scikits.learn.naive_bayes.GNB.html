scikits.learn.naive_bayes.GNB¶

class scikits.learn.naive_bayes.GNB¶

Gaussian Naive Bayes (GNB)

Parameters :

X : array-like, shape = [n_samples, n_features]

Training vector, where n_samples in the number of samples and n_features is the number of features.

y : array, shape = [n_samples]

Target vector relative to X

Examples

>>> import numpy as np
>>> X = np.array([[-1, -1], [-2, -1], [-3, -2], [1, 1], [2, 1], [3, 2]])
>>> Y = np.array([1, 1, 1, 2, 2, 2])
>>> from scikits.learn.naive_bayes import GNB
>>> clf = GNB()
>>> clf.fit(X, Y)
GNB()
>>> print clf.predict([[-0.8, -1]])
[1]

Attributes

proba_y array, shape = [n_classes] probability of each class.
theta array, shape [n_classes * n_features] mean of each feature for the different class
sigma array, shape [n_classes * n_features] variance of each feature for the different class
